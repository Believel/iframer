进阶功能

整个访问过程中，主页面一直保持在可用状态，我们可以做的事情就很多了。

发挥你的想象，把音乐播放器做在主页面里，还有就是IM聊天功能。是不是想想就很棒呢！


iframer.js初始化接收的全部参数


iframer.init({
    //存放iframe的容器
    container : document.getElementById('sap-container'),
    //默认页面
    default_url:'index.html',
    //不处理跳转的a链接class
    expect_class : 'spa-expect-links',
    //当每次准备更改页面标题时回调
    beforeTitleChange : function(title){
      return 'iframer-' + title
    },
    //iframe 载入事件
    iframeOnload: function(iframw_window,iframe_document){
       //
    }
});

主动跳转至某一页面


//跳转到某个页面
iframer.jumpTo('/article/id/342');
//若在iframe中执行,请带上window参数（用于计算相对路径）
top.iframer.jumpTo('area/tags',window);

修改页面标题

修改时会触发beforeTitleChange事件，执行相应修改。


iframer.updateTitle('欢迎光临');
